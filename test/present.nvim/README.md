# `present.nvim`

This is a presentation plugin for Neovim!!!

## Usage

You just use it!

```lua
print "Hello..."
print "...from Lua!"
```

```lua
print "Another hello from Lua!"
```

```js
const hello = "Hello world!";
console.log(hello);
console.log("My object:", { name: "John", age: 100 });
```

```python
hello = "Hello from python!"
print(hello)
```
